# RedMattina Website

Portfolio site built with Next.js App Router, TypeScript, Tailwind CSS, and a `react-pageflip` sketchbook experience.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

```bash
npm run dev
npm run build
npm run lint
```

## Project Structure

- `src/app`: routes, route layouts, route-level styles.
- `src/components`: reusable UI components.
- `src/features`: feature-specific interactive components.
- `src/data`: site content and configuration.
- `public`: static images and icons.
